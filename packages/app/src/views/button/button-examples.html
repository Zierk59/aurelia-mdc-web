Trailing icon

Certain icons make more sense to appear after the button's text label rather than before. This can be accomplished by putting the icon markup after the element with the mdc-button-label attribute.

NOTE: The mdc-button-label attribute is required in order for the trailing icon to be styled appropriately.

Accessibility

Material Design spec advises that touch targets should be at least 48 x 48 px. To meet this requirement, add the `touch` property.

Note that the outer mdc-touch-target-wrapper element is only necessary if you want to avoid potentially overlapping touch targets on adjacent elements (due to collapsing margins).
